1. OBJETO DEL SOFTWARE
--------------------------------------------------------------------------------

Gestión de Proveedores es una aplicación de escritorio destinada a negocios
gastronómicos que permite:

  - Administrar catálogos de productos e insumos.
  - Registrar y gestionar proveedores con sus condiciones comerciales.
  - Importar y procesar listas de precios en formato Excel o PDF.
  - Comparar precios entre proveedores con cálculo de impuestos (IVA, IIBB).
  - Simular órdenes de compra y facturas proforma.
  - Utilizar inteligencia artificial para la categorización y equivalencia
    automática de productos (función opcional, requiere clave API de OpenAI).

El software no tiene conexión obligatoria a internet. Toda la información
ingresada por el usuario se almacena exclusivamente en su equipo local
mediante una base de datos SQLite.
